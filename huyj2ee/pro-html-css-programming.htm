I. Css layout
1. Position properties
There are 4 position properties top, bottom, left, and right to position an HTML element.
2. Position type property (property position)
1. static: default position, element is positioned by normal flow of page and 4 position properties values is ignored.
2. relative: shift the element by values of position properties to its normal position.
3. fixed: positioned relative to the viewport.
4. absolute: positioned relative to the nearest positioned ancestor.
5. sticky: positioned based on the user's scroll position.

III. Css animation
1. Animation properties
They are:
animation-name: reference to applied animation.
animation-duration: determine animation speed, it is the duration that animation will demonstrate. Setting 0 results in animation disabled.
animation-delay: delay for the start of an animation.
Negative value of animation-delay results in animation started as if it had already been playing for N seconds.
animation-iteration-count: T.B.D.
animation-direction: whether an animation should be played forwards, backwards or in alternate cycles.
normal: the animation is played as normal (forwards). This is default.
reverse: the animation is played in reverse direction (backwards).
alternate: the animation is played forwards first, then backwards.
alternate-reverse: the animation is played backwards first, then forwards.
animation-fill-mode: T.B.D.
animation: animation shorthand.
Default:
div {
     animation-name: example;
     animation-duration: 5s;
     animation-timing-function: linear;
     animation-delay: 2s;
     animation-iteration-count: infinite;
     animation-direction: alternate;
}
Shorthand:
div {
     animation: example 5s linear 2s infinite alternate;
}
