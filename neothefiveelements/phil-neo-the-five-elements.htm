The Five Elements is a fivefold conceptual scheme that many traditional Chinese fields used to explain a wide array of phenomena, from cosmic cycles to the interaction between internal organs. It is an ancient though, this blog entry tries to model the conceptual scheme into mathematics graph (graph theory) so that it is friendly with modern science and names it as Neo The Five Elements. This Neo The Five Elements then could be applied to model some other modern science fields.












I. Genuine Fivefold conceptual scheme
The genuine though is based on two concepts: phases or elements and cycles.
There are five phases/elements:
Spring/Wood: a period of growth, which generates abundant wood and vitality
Summer/Fire: a period of swelling, flowering, brimming with fire and energy
Earth: the in-between transitional seasonal periods, or a separate 'season' known as Late Summer or Long Summer – in the latter case associated with leveling and dampening (moderation) and fruition
Metal/Autumn: a period of harvesting and collecting
Water/Winter: a period of retreat, where stillness and storage pervades
Cycles: there is two cycles
A generating or creation (生, shēng) cycle, also known as "mother-son"
Wood feeds Fire
Fire creates Earth (ash)
Earth bears Metal
Metal collects Water
Water nourishes Wood
An overcoming or destruction (剋/克, kè) cycle, also known as "grandfather-grandson"
Wood parts Earth (such as roots or trees can prevent soil erosion)
Earth dams (or muddies or absorbs) Water
Water extinguishes Fire
Fire melts Metal
Metal chops Wood
II. Neo The Five Elements
1. Modeling Genuine Fivefold conceptual scheme into a mathematics graphs is straightforward.
Elements is modeled as numbers of range [0..4].
Elements = {0, 1, 2, 3, 4}
0: presents water element
1: presents wood element
2: presents fire element
3: presents earth element
4: presents metal element
Cycles: there are two cycle, each cycle is modeled as a mathematics circular graph that vertex set is Elements.
1. Creation cycle
Graph definition: Creation = (Elements, Ec)
Where Ec = {(0, 1), (1, 2), (2, 3), (3, 4), (4, 0)}
(0, 1): Water nourishes Wood
(1, 2): Wood feeds Fire
(2, 3): Fire creates Earth (ash)
(3, 4): Earth bears Metal
(4, 0): Metal collects Water
2. Overcoming cycle
Graph definition: Overcoming = (Elements, Eo)
Where Eo = {(0, 2), (1, 3), (2, 4), (3, 0), (4, 1)}
(0, 2): Water extinguishes Fire
(1, 3): Wood parts Earth (such as roots or trees can prevent soil erosion)
(2, 4): Fire melts Metal
(3, 0): Earth dams (or muddies or absorbs) Water
(4, 1): Metal chops Wood

2. Properties of Neo The Five Elements graphs
Creation cycle:
Ec = {(0, 1), (1, 2), (2, 3), (3, 4), (4, 0)}
Ec = { (e1, e2) | (e1 ∈ Elements) ∧ (e2 ∈ Elements) ∧ e2 = (e1 + 1) mod 5 }
Overcoming cycle:
Eo = {(0, 2), (1, 3), (2, 4), (3, 0), (4, 1)}
Eo = { (e1, e2) | (e1 ∈ Elements) ∧ (e2 ∈ Elements) ∧ e2 = (e1 + 2) mod 5 }
3. Generic Neo The Five Elements graphs
Base on properties of Neo The Five Elements graphs, we could build a generic Neo The Five Elements graphs as below:
Given 道: Tao = {0}
Given 兩極: LoengGit = {-1, 1}
Given 三才: SamChoi = LoengGit ⋃ Tao = {-1, 0, 1}
Where:
天: 1
人: 0 (People is born from Tao and endups life with returnning to Tao)
地: -1
Given 兩儀: LoengJi = {X, Y} where: X = Y = SamChoi
Given N = {0, 1, 2, 3, 4} We build a mapping as below:
f: X x Y → N
f(x, y) = 3 if |x| = |y|
Otherwise:
f(-1, 0) = 4
f(0, 1) = 0
f(1, 0) = 1
f(0, -1) = 2
We build a relation R on X x Y as below:
(x1, y1) R (x2, y2) ⇔ f(x1, y1) = f(x2, y2)
This R is an equivalence relation and it partitions X x Y as a family of classes:
X x Y = [3] ⋃ [4] ⋃ [0] ⋃ [1] ⋃ [2]
Base on family of classes, we define:
Elements = {0, 1, 2, 3, 4}
As properties of Neo The Five Elements graphs, we have
Creation = (Elements, Ec)
Ec = { (e1, e2) | (e1 ∈ Elements) ∧ (e2 ∈ Elements) ∧ e2 = (e1 + 1) mod 5 }
Overcoming = (Elements, Eo)
Eo = { (e1, e2) | (e1 ∈ Elements) ∧ (e2 ∈ Elements) ∧ e2 = (e1 + 2) mod 5 }
So:
GenericNTFE = (Creation, Overcoming)
