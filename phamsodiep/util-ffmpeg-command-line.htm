FFmpeg is a free and open-source project for handling video, audio, and other multimedia files and streams. It supports a CUI application (console command) for multimedia files editing, also a codec library used for software developing. This blog entry shares some usage guideline of the console command one.












I. Installation
1. For Debian linux
apt install ffmpeg
2. For Windows
Download bundle at website: https://www.ffmpeg.org (something like this: https://ffmpeg.zeranoe.com/builds/win32/static/ffmpeg-20200312-675bb1f-win32-static.zip )
Extract zip file and update Windows PATH environment variable point to folder ffmpeg\bin
II. Usecase or multimedia files operations
1. Simple codec/format convert
C:\Users\Guest>ffmpeg -i LiuLangGe.m4a LiuLangGe.mp3
C:\Users\Guest>ffmpeg -i LiuLangGe.mp4 LiuLangGe.mkv
2. Codec/format convert
C:\Users\Guest>ffmpeg -i LiuLangGe.mp4 -c:v libx265 -c:a copy LiuLangGe.mkv
☞ This will convert input file LiuLangGe.mp4 to output file LiuLangGe.mkv with video encoder codec libx265 and audio encoder codec is same with input audio codec (copy)
3. Multimedia files cut (extract a time duration of source file)
C:\Users\Guest>ffmpeg -i AndyLauVol234.mp4 -ss 00:01:00 -to 00:04:00 Song1.mp4
☞ This will extract Song 1 which lasts from 00:01:00 to 00:04:00 of the full volume file AndyLauVol234.mp4 results in a file name Song1.mp4 of duration lengh of 3 minutes.
